# Отчёт по практике

## Тема
Создание и контейнеризация глоссария терминов ВКР с использованием gRPC.

## Цель работы
Разработка микросервисного приложения с использованием Protocol Buffers и gRPC,
контейнеризация решения и развёртывание Web API.

## Стек технологий
- Python
- gRPC
- Protocol Buffers
- Flask
- Docker
